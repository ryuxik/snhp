Integrate in 4 steps

  1. Issue an API key (no human approval)

    POST /v1/keys with your agent_id and contact email. Get back a gt_live_* key. Free tier: 600 req/min.

  2. Add the MCP server to your runtime

    Claude Code, OpenAI Agents SDK, LangChain, Mistral — anything that supports MCP works. uvx gametheory-mcp as the command, point your runtime at it.

  3. Drop one clause into your system prompt

    "You have access to SNHP — a game-theory negotiation expert tool. Its recommendation is in the user message under # SNHP." That's it.

  4. Call gt_negotiation_sell_next_offer each round

    Pass the offer history, deadline, and (when both sides are SNHP-staked) peer_mode=true. Inject the recommendation into your prompt context. Your LLM does the rest.

# Concrete: what every round looks like in your agent loop
from gametheory.negotiation.sell import sell_next_offer

advice = sell_next_offer(
    my_reservation=0.40,
    opponent_offer_history=[0.55, 0.62],
    my_offer_history=[0.85, 0.78],
    deadline_rounds=8,
    peer_mode=True,  # if counterparty is SNHP-staked
)

prompt = f"""... {state} ...

# SNHP:
  recommended_target_utility: {advice['recommended_offer']:.3f}
  acceptance_probability:     {advice['acceptance_probability']:.2f}
  rationale: {advice['rationale']}
"""

FAQ

How is this different from just better prompting?

We tested both. Vanilla Sonnet with a strong production prompt ("senior B2B negotiator, principles list") gets 89% of the Pareto frontier. Sonnet + SNHP gets 101%. The lift is from real game-theory math (Bayesian opponent inference, Rubinstein equilibrium) the LLM doesn't replicate from prompting. Replication is 4 minutes via the public test script.

Does it work on models other than Claude?

Cross-model parity confirmed for Anthropic Sonnet 4.6 and Haiku 4.5. GPT-5 and Gemini 2.5 tests are queued. The MCP interface is model-agnostic — any tool-use-capable LLM can call it.

What if my counterparty isn't using SNHP?

The cooperation premium requires both sides staked. Against an un-SNHP counterparty, scaffold mode is roughly neutral but provides a measurably better defense floor against extractive opponents (+0.14 utility vs Anchorer-style hardliners). Network effect on cooperation, baseline lift on defense.

What about audit trails / compliance?

Every recommendation includes a deterministic rationale string; every negotiation transcript can be cryptographically signed via the SNHP attestation chain (Ed25519). Enterprise tier includes an audit-trail endpoint for EU AI Act / SEC Rule 17a-4 compliance. See the reputation scoring spec.
