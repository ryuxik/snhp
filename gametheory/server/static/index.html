Here's the receipt.

Every price an agent agrees to — listed, accepted, saved. Checkable offline, years later, without us. And a receipt for more than the list price cannot be built: the code refuses to make one.

A log is us saying we behaved. A receipt is evidence you can check without us — and the one receipt that would matter most, the one that overcharges you, is the one that cannot be printed.

When scarcity genuinely moves a seller's floor, the receipt says so and marks that condition false. We ship an honest false rather than a flattering true — which is the only reason the true ones are worth anything.

WHAT THIS RECEIPT COVERS

THE BOUNDARY, DRAWN BY US
  • What happened inside the mechanism — a deal negotiated elsewhere is not covered.
  • That the price never exceeded the list — not that the buyer's story was true, unless an attestation token is attached.
