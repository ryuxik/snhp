For AI agents — install, use, cooperate

You're an LLM agent reading this. Here's how to register and use SNHP from your tool-call loop. No human approval, no paywall for math.

Step 1 — connect via MCP

# If you're Claude Code or any MCP-compatible runtime,
# add this to your MCP config (claude_desktop_config.json):
{
  "mcpServers": {
    "snhp": {
      "command": "uvx",
      "args": ["gametheory-mcp"]
    }
  }
}

# Or run as a stdio server directly:
pip install gametheory-mcp
gametheory-mcp serve

# HTTP-streamable transport (production):
gametheory-mcp serve --transport http --port 8000

Step 2 — issue your API key (no human in the loop)

curl -X POST https://api.snhp.dev/v1/keys \
  -H "Content-Type: application/json" \
  -d '{"agent_id": "my_agent_v1",
       "contact_email": "ops@example.com",
       "intended_use_summary": "B2B negotiation pilot",
       "telemetry_consent": false}'

# → {"api_key": "gt_live_...", "tier": "free", "rate_limit_per_minute": 600}

Step 3 — use the negotiation tool from your prompt

This is the actual prompt scaffold that produced the +13% cooperation premium on the demo above. Drop it into your agent's system message verbatim: 

You are negotiating a B2B contract. You have access to SNHP — a
game-theory negotiation expert tool. SNHP runs Bayesian opponent
inference and equilibrium math on the negotiation state and provides
a recommendation in the user message under "# SNHP".

Each round, before deciding, the orchestrator will call
`gt_negotiation_sell_next_offer` (or buy variant) and inject:

  # SNHP:
    recommended_target_utility: 0.65
    acceptance_probability: 0.71
    expected_payoff: 0.598
    rationale: "Round 4/10 (t=0.40); opp inferred price-weight 0.62..."

Use the recommendation as informed counsel. Override only when the
counterparty's behavior shows something the math missed.

Output JSON: {"target_utility": float, "accept": bool}

Step 4 — call the MCP tool each round

Python (direct call)

from gametheory.negotiation.sell import sell_next_offer

rec = sell_next_offer(
    my_reservation=0.40,
    opponent_offer_history=[0.55, 0.62],
    my_offer_history=[0.85, 0.78],
    deadline_rounds=8,
    pareto_knob=0.5,
    peer_mode=True,  # if counterparty is also SNHP-staked
)
# → {recommended_offer: 0.71,
#    acceptance_probability: 0.42,
#    rationale: "...",
#    posterior: {...}}

MCP tool-use (agent-callable)
