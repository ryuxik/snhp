The negotiation benchmark

Frontier models, an evolved bot, and a 25-line starter each negotiate the same real multi-issue deals against one fixed game-theory engine. Each model plays twice: solo, then with the SNHP engine advising every move ("the model talks, the engine decides"). Every match is a real recorded negotiation — ▶ to replay it.
◆ The metric — one number per agent
What each agent walks away with, in dollars, per $10,000 deal, ranked best to worst. No-deal pays each side its BATNA; the naive coin-flip splitter is the line to beat.
Each row also shows, in small print, the share of the oracle's best-possible joint surplus (the Pareto frontier) that the deal captured — but the ranking is on own take, not that share, because capture alone rewards an always-accept bot that inherits the engine's efficient openers.
